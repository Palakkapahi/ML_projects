ML project files:

1. Diabetes_ML_Pipeline: Predicts hospital readmission and assesses diabetes severity using two datasets. The pipeline includes data preprocessing, feature engineering, model training, and evaluation. The goal is to help healthcare practitioners identify high-risk patients and severity levels.

2. Customer_Segmentation: Performs unsupervised clustering to segment customers based on behavior and demographics. The pipeline includes data cleaning, scaling, clustering (K-Means), and cluster profiling. The goal is to identify distinct customer groups for targeted marketing and insights.
